# 👑 Kohinoor Ventures League — JBN LT05

Live stock-market style business networking dashboard.

## Deploy to Vercel (3 steps)

1. Push to GitHub
2. Import repo on vercel.com
3. Deploy — done!

## Default Credentials
- Admin: `admin / kvl@2025`
- Viewer: `viewer / view123`  
- Participants: `harshit.shah / apx001`, `nikesh.shah / smt001`, etc.

## Stack
- Node.js + Express backend
- Vanilla JS + Chart.js frontend
- No database needed (in-memory, resets on redeploy)
- PWA installable
